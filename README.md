The Notes App is a simple and intuitive application designed for creating, 
managing, and organizing your notes seamlessly.
Building a Notes App is a great first step for beginners as
it covers essential concepts of Android development.

Features 📱
Create and Edit Notes
Delete Notes
Organize with Categories
Offline Access

Concepts Used ⚙️
Android Jetpack
Clean Architecture
Navigation Components
RoomDB
Coroutines
